## 面向对象

> 简介

```go
1 go 没有面向对象语法等的要求
2 go 语言对于面向对象的设计非常简单而优雅
3 没有封装(数据更加安全)、继承(减少代码冗余)、多态(可以产生不同的实例)的概念，但是通过自定义也可以实现
封装：可以通过方法实现
继承：可以通过匿名字段实现
多态：可以通过接口实现
```

> 匿名字段

```go
1 go 支持只提供类型而不写字段名的方式，也就是匿名字段，也称为嵌入字段
2 同名字段情况
3 所有的内置类型和自定义类型都是可以作为匿名字段去使用
4 指针类型的匿名字段
// 1 
package main

   age  int
}
```

> 方法

```go
1 在面向对象编程中，一个对象其实也就是一个简单的值或者一个变量，在这个对象中会包含一些函数
2 这种带有接收者的函数，我们称为方法，本质上，一个方法则是一个和特殊类型关联的函数
3 方法的语法如下：
	func (接收参数名 接收类型) 方法名(参数列表)(返回值)
4 可以给任意自定义类型（包括内置类型，但不包括指针类型）添加相应的方法
5 为类型添加方法
/////// 基础类型作为接收者
package main

import "fmt"

// 基础类型作为接收者
// 任务：定义方法，实现2个数相加

// 自定义
type MyInt int

func Add(a, b MyInt) MyInt {
   return a + b
}

// 面向对象
func (a MyInt) Add(b MyInt) MyInt {
   return a + b
}

func main() {
   var a MyInt = 1
   var b MyInt = 2
   // 面向过程的调用方法
   fmt.Println("Add(a,b) = ", Add(a, b))
   // 面向对象的调用
   fmt.Println("a.Add(b) = ", a.Add(b))
}
/////// 结构体作为接收者
package main

import "fmt"

// 结构体作为接收者
type Person struct {
	name string
	sex  string
	age  int
}

// 给Person添加个方法，方法名就是PrintInfo
func (p Person) PrintInfo() {
	fmt.Println(p.name, p.sex, p.age)
}

func main() {
	p := Person{"zs", "male", 18}
	p.PrintInfo()
}
```

### 面向对象练习

```go
/*
实现一个简单的学生管理系统，每个学生有分数、年级、性别、名字等字段，用户可以在控制台输入1添加学生、输入2修改学生信息、输入3打印所有学生列表，输入4结束程序，如下：
1 添加学生
2 修改学生信息
3 打印所有学生列表
4 结束程序
*/
Student.go
package main

// 定义学生对象
type Student struct {
   Username string
   Sex      int
   Score    float32
   Grade    string
}

// 构造方法
func NewStudent(username string, sex int, score float32, grade string) (stu *Student) {
   stu = &Student{
      Username: username,
      Sex:      sex,
      Score:    score,
      Grade:    grade,
   }
   return
}
main.go
package main

import (
   "fmt"
   "os"
)

var (
   AllStudents []*Student
)

// 循环打印帮助信息
func showMenu() {
   fmt.Println("1.添加学生")
   fmt.Println("2.修改学生")
   fmt.Println("3.打印学生")
   fmt.Println("4.结束程序")
}

// 添加学生的方法
func AddStudent() {
   stu := inputStudent()
   // 假设没有重名的学生，名字若一样，则做更新
   for index, v := range AllStudents {
      if v.Username == stu.Username {
         // 做更新
         AllStudents[index] = stu
         fmt.Println("更新成功")
         return
      }
   }
   // 不重名直接加
   AllStudents = append(AllStudents, stu)
   fmt.Println("学生插入成功")
}

// 用户输入的方法
func inputStudent() *Student {
   // 定义输入的接收变量
   var (
      username string
      sex      int
      score    float32
      grade    string
   )
   fmt.Println("请输入学生姓名：")
   _, _ = fmt.Scanf("%s\n", &username)
   fmt.Println("请输入学生性别：[0|1]")
   _, _ = fmt.Scanf("%d\n", &sex)
   fmt.Println("请输入学生分数：[0-100]")
   _, _ = fmt.Scanf("%f\n", &score)
   fmt.Println("请输入年级：")
   _, _ = fmt.Scanf("%s\n", &grade)
   // 创建对象
   stu := NewStudent(username, sex, score, grade)
   return stu
}

// 修改学生
func ModifyStudent() {
   stu := inputStudent()
   // 假设没有重名的学生，名字若一样，则做更新
   for index, v := range AllStudents {
      if v.Username == stu.Username {
         // 做更新
         AllStudents[index] = stu
         fmt.Println("更新成功")
         return
      }
   }
   fmt.Println("学生插入成功")
}

// 打印学生列表
func ShowAllStudent() {
   // 遍历切片打印
   for _, v := range AllStudents {
      fmt.Printf("学生:%s 信息: %#v\n", v.Username, v)
   }
   fmt.Println()
}

// 写主函数
func main() {
   for {
      // 1.展现信息
      showMenu()
      // 2.接收用户输入
      var i int
      _, _ = fmt.Scanf("%d\n", &i)
      switch i {
      case 1:
         AddStudent()
      case 2:
         ModifyStudent()
      case 3:
         ShowAllStudent()
      case 4:
         os.Exit(0)
      }
   }
}
```
